README_mac.txt for version 8.2 of Vim: Vi IMproved.

This file explains the installation of Vim on macOS systems.
See "README.txt" for general information about Vim.

To install Vim from source:

  1. Download the source code from https://github.com/vim/vim.

  2. Follow the instructions in src/INSTALLmac.txt.

To install Vim with Homebrew:

  1. Install Homebrew from https://brew.sh/.

  2. Run:

         brew install vim
