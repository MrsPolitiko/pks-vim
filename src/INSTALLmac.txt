INSTALLmac.txt - Installation of Vim on Mac

This file contains instructions for compiling Vim. If you already have an
executable version of Vim, you don't need this.

Classic Mac OS is no longer supported. If you really want it, use Vim 6.4.

1.1 MacVim

1.2 Homebrew

  To install Vim with Homebrew, see Homebrew's website for instructions.

1.3 Source

  To install Vim from source:

    1. Install Xcode or Command Line Tools.

    2. Use the standard Unix routine:

         cd .../src
         make
         sudo make install

  To run the tests, first install GNU Libtool with the program prefix "g".
